Installation

Asset Library (Free, Stable, Recommended)

  1. In Godot, open the AssetLib tab.
  2. Search and select “All Projectiles”.
  3. Download and install the addon.


Patreon (Latest, New Features, with Demo Scenes)

  1. Follow the link to my Patreon.
  2. Download the latest release build. (It’s free for everyone and comes with all the new features and Demo Scenes)
  3. Extract the zip file and move the addons/all_projectiles folder into your project root location.


GitHub Release (Free, Stable)

  1. Download the latest release build.
  2. Extract the zip file and move the addons/all_projectiles folder into your project root location.


Itchio (Paid, Latest, Future Updates, with Demo Scenes)

  1. Follow the link to my Itch.io.
  2. Purchase and download the tool.
  3. Extract the zip file and move the addons/all_projectiles folder into your project root location.
